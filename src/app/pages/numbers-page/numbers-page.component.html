Pipes numericos

Algunos pipes numericos reciben atributos para personalizar

Esta notacion significa, queremos el numero entero de 2 a 4 decimales: "1.2-4"

Number: 1.2-2
{{ totalSells() | number }}
{{ totalSells() }}
Number with args
{{ totalSells() + 300 | number : '1.2-2' }}
{{ totalSells() }}
Currency
{{ totalSells() | currency : 'PEN' : 'symbol-narrow' }}
{{ totalSells() }}
Percents
{{ percent() | percent : '1.2-2' }}
{{ percent() }}
